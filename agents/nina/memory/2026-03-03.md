### Architecture technique proposée

**Stack :**
1. **Front-end** : Emergent
   - Interface calendrier + réservation
   - Pages logements (fiches détaillées)
   - Formulaire réservation + paiement
   - Espace client (optionnel phase 1)

2. **Backend** : Supabase
   - Base de données (logements, réservations, disponibilités)
   - Auth (espace client)
   - API REST

3. **Intégrations** :
   - Stripe (paiement sécurisé)
   - Airbnb API ou iCal (sync dispos)
   - Resend/SendGrid (emails confirmation)
