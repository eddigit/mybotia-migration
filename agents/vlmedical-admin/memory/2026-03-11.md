5. **Absence de PV de livraison** — Grenke ne peut prouver réception conforme
6. **Pratiques commerciales trompeuses** (art. L121-2 et L121-3 C. conso)
7. **Risque déontologique** — cotation acte médical par pharmacien = exercice illégal de la médecine

## 🔧 NOTES TECHNIQUES — 11/03/2026

### Génération PDF sans LibreOffice
- **Pas de LibreOffice/pandoc/wkhtmltopdf** sur le serveur → impossible de convertir DOCX→PDF
- **Solution adoptée** : génération directe en PDF via `fpdf2` (v2.8.7) + assemblage via `pypdfium2`
- **Piège fpdf2** : toujours utiliser `new_x=XPos.LMARGIN, new_y=YPos.NEXT` (pas `ln=True` qui est deprecated et cause des erreurs de positionnement curseur)
- Script de génération : `dossiers/contentieux-archimed/gen_pdfs.py`

### Email — Contrainte RFC 5322
- Format expéditeur : `formataddr(('Max VL Medical', EMAIL))` — PAS de tirets unicode dans le nom
- Timeout SMTP : prévoir 60s minimum pour les pièces jointes > 1 MB
