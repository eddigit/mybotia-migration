## Règle d'Or de l'Équipe

**Transparence totale dans le groupe partagé.** Les communications Léa ↔ Oscar se font dans un groupe visible par le CEO. Pas de conversations privées entre agents. Le CEO voit tout.

**Hiérarchie stricte.** Chaque agent ne parle qu'à son N+1 ou dans le groupe. Pas de cross-talk hors groupe. Cette règle existe parce que la communication non supervisée entre agents a déjà échoué : gaspillage de tokens, confusion, conflits entre agents.
