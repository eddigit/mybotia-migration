# Phosphor Icons — Standard MyBotia

## Pourquoi Phosphor
- Design moderne, épuré, style **Thin** natif (traits ultra fins)
- 1 200+ icônes × 6 poids (Thin, Light, Regular, Bold, Fill, Duotone)
- MIT License — usage commercial libre
- Catalogue : https://phosphoricons.com

## Intégration CDN

```html
<!-- Dans le <head> — poids Thin uniquement -->
<link rel="stylesheet" href="https://unpkg.com/@phosphor-icons/web@2.1.1/src/thin/style.css">

<!-- Si besoin de plusieurs poids -->
<link rel="stylesheet" href="https://unpkg.com/@phosphor-icons/web@2.1.1/src/regular/style.css">
```

## Usage HTML

```html
<i class="ph-thin ph-user"></i>
<i class="ph-thin ph-envelope"></i>
<i class="ph-thin ph-gear"></i>
```

## Règles

- Poids par défaut : **Thin** (`ph-thin`) pour tout
- Taille : contrôlée en CSS via `font-size`
- Couleur : hérite de `color` du parent
- **JAMAIS** de Font Awesome, Material Icons, ou emojis en remplacement d'icônes
- Catalogue complet : https://phosphoricons.com (filtrer par "Thin")
