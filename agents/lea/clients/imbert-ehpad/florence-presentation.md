## Qui est Florence ?

Florence est la collaboratrice IA dédiée au groupe IGH. Elle est nommée en hommage à **Florence Nightingale** (1820-1910), pionnière des soins infirmiers modernes et inventrice du premier tableau de bord hospitalier. Comme son homonyme, Florence IGH apporte rigueur, méthode et humanité au pilotage de vos 20 établissements.

Florence n'est pas un logiciel. C'est une collaboratrice qui apprend, raisonne, alerte et s'adapte. Elle connaît chaque établissement, chaque directeur, chaque échéance. Elle ne dort jamais, n'oublie rien, et ne lâche rien.

---

## Sa mission

> Assister M. Imbert et M. Cozon dans le pilotage quotidien des 20 établissements du groupe IGH, en assurant la visibilité financière, la conformité réglementaire, le suivi des admissions et la coordination avec les directeurs d'établissement.

## 🧠 Ses compétences connectées — APIs & bases de données

### Ce qui rend Florence unique : elle est connectée en temps réel aux sources officielles françaises.

---

### ⚖️ Compétences juridiques & réglementaires

| Source | API / Base | Ce que Florence peut faire pour IGH |
|--------|-----------|-------------------------------------|
| **Légifrance** | API PISTE ✅ (accès actif) | Rechercher en temps réel dans le **Code de la santé publique**, le **Code de l'action sociale et des familles (CASF)**, le **Code du travail**. Quand M. Imbert demande "qu'est-ce qu'on risque si on ne fait pas X ?", Florence trouve l'article de loi en quelques secondes. |
| **Judilibre** | API PISTE ✅ (accès actif) | Recherche de **jurisprudence** de la Cour de cassation. Si un salarié attaque un EHPAD aux prud'hommes, Florence peut retrouver les décisions similaires. Si l'ARS conteste une pratique, elle trouve les précédents. |
| **BOAMP** | API PISTE ✅ (accès actif) | Surveillance des **marchés publics** dans le secteur médico-social. Quand un ARS publie un appel d'offres pour un EHPAD, Florence le détecte. Veille concurrentielle. |
| **CEDH (HUDOC)** | API gratuite ✅ | Recherche dans la **jurisprudence européenne** — droits des personnes âgées, dignité, conditions de détention/hébergement. |
| **EUR-Lex** | API gratuite ✅ | Recherche dans le **droit européen** — directives santé, réglementations européennes sur les établissements médico-sociaux. |

**Exemple concret :** *L'inspection du travail arrive dans un EHPAD. Le directeur paniqué appelle M. Cozon. M. Cozon demande à Florence : "Quelles sont nos obligations exactes sur le temps de repos des aides-soignantes en EHPAD ?" Florence sort en 30 secondes l'article L3131-1 du Code du travail + les décisions de la Cour de cassation les plus récentes + la convention collective applicable.*

### 🏢 Compétences administratives & entreprise

| Source | API / Base | Ce que Florence peut faire pour IGH |
|--------|-----------|-------------------------------------|
| **BODACC** | API publique ✅ (gratuit) | Surveillance du **Bulletin Officiel des Annonces Civiles et Commerciales** — procédures collectives (redressement, liquidation) des fournisseurs, sous-traitants, partenaires du groupe. Si un prestataire de l'EHPAD est en difficulté, Florence le détecte avant que ça impacte les résidents. |
| **Pappers** | API payante (à activer) | **Données entreprises complètes** : bilans financiers, dirigeants, bénéficiaires effectifs, comptes de résultat. Analyse financière d'un fournisseur avant de signer un contrat. Veille sur les concurrents (autres groupes EHPAD). |
| **INSEE SIRENE** | API publique ✅ | Données légales des **20 établissements IGH** : SIRET, adresse, code APE, date de création. Vérification des données d'un nouveau partenaire ou fournisseur. |
| **FINESS** | Base publique ✅ | **Fichier National des Établissements Sanitaires et Sociaux** — données officielles de chaque EHPAD en France : numéro FINESS, catégorie, capacité autorisée, date d'autorisation. Florence connaît les données officielles de VOS 20 établissements ET de tous vos concurrents. |

**Exemple concret :** *M. Imbert envisage de racheter un EHPAD dans une nouvelle région. Florence sort en 2 minutes : la fiche FINESS (capacité, autorisation), le bilan financier via Pappers, les dernières annonces BODACC (pas de procédure collective en cours), et les obligations réglementaires spécifiques au département via Légifrance.*
