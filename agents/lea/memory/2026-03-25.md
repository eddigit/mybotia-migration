## Architecture vocale — Recherche approfondie (04h57)

**3 approches fiables identifiées par Gilles :**

1. **Pipeline Cascadé Streaming** (notre baseline POC) :
   - STT (Deepgram Nova-3 ou Whisper/Groq) → LLM (Claude) → TTS (edge-tts Vivienne)
   - TTFA ~755ms — quasi instantané pour l'humain
   - Architecture de référence (Vapi, Retell, Bland.ai l'utilisent)

2. **Pipeline Hybride 2 étapes** (veille, pas pour le POC) :
   - Audio → Qwen3-Omni (traite audio directement) → TTS
   - TTFA ~736ms, function calling natif — dépend de vLLM, pas Anthropic

3. **Détection de tour sémantique** (indispensable) :
   - LiveKit a ça en natif — confirme le choix
   - Analyse le sens, pas juste le silence — critique pour avocats

**Compléments validés :** Barge-in (LiveKit natif), Wake word Picovoice Porcupine (local), Serverless pour scaling
