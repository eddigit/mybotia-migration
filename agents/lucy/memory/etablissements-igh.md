# Cartographie des établissements IGH

## Types d'établissements
- EHPAD classiques (hébergement permanent/temporaire, valides à dépendants)
- EHPAD avec unité protégée Alzheimer (déambulation sécurisée, activités thérapeutiques)
- Résidences pour personnes encore autonomes (parfois couples)
- Établissements mixtes : EHPAD + SSR + USLD
